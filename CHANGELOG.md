# Changelog

All notable changes to Kikoku will be documented in this file.

## [2.0] - 2024

### Added
- Initial public release
- 60+ security audit checks
- ACL traversal with BloodHound-style path analysis
- Standalone LDAP implementation (no ActiveDirectory module required)
- Beautiful formatted output with ASCII art banner
- Comprehensive AD security analysis
- Attack path discovery through nested group memberships
- Support for all major AD security checks

### Features
- Domain enumeration and analysis
- User, Group, Computer enumeration
- Password policy analysis
- Trust and delegation analysis
- Kerberoastable/AS-REP roastable detection
- Shadow admins detection
- DCSync shadow permissions
- GPO delegation analysis
- Tiered administration model checks
- Kerberos/LDAP hardening analysis
- Exchange security analysis
- ADCS vulnerability detection
- SCCM/MECM security analysis
- Cloud identity integration analysis
- And 40+ more security checks

### Technical
- Raw LDAP queries via System.DirectoryServices
- SDDL parsing and ACL analysis
- Recursive group membership resolution
- Automatic credential handling
- Color-coded severity ratings
- Professional output formatting
